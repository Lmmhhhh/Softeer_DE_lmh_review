# Predicting Election Results from Twitter Using Machine Learning Algorithms summary 소감문

트위터 게시글을 모아 감정을 분석하고 선거 결과를 예측해 94.2%의 정밀도를 보였다는 논문 summary를 읽어보고 주제도 신선하고 재미있다고 느꼈다. 
하지만 몇 가지 의문이 들었다. 
가장 걸리는 것은 트위터에 적극적으로 글을 올리는 사람 소위 "트위터리안"이라 불리는 사람들이 곧 실제 유권자 전체는 아니라는 점이다. 전체 유권자 중 트위터에 적극적으로 정치적 목소리를 내는 유권자는 극소수일 것이라는 생각이 들었다. 그렇기 때문에 그 표본이 진짜 민심을 대표하는지에 대해 의문이 들었다. 물론 논문 전문을 읽어본 것은 아니지만, 논문에 해당 내용이 설명되어 있지 않다면 고려해봐야할 부분인 것 같다. 
또한, 반어법이나 비꼬는 표현(영어로 sarcasm), 광고성 도배글 같은 게시글들을 잘 걸러냈을까하는 생각이 들었다.
그럼에도 사람들이 실시간으로 쏟아내는 목소리를 여론 파악에 활용한다는 발상 자체는 신선하고 흥미로웠다. 다만, 그 표본이 정말 대표성이 있는지 그리고 어떤 글을 어떻게 걸러냈는지가 더 궁금해졌던 summary였다.
w2m5 미션 과정에서 kaggle에서 제공된 트위터 감정 데이터셋(Sentiment140)을 사용하면서 이미 긍정/부정 라벨이 붙어있는 데이터를 그대로 가져다 썼지만, 정작 그 라벨이 어떻게 붙은건지는 깊이 생각해보지 않았던거 같다. 이 부분은 추후에 조사를 시도해봐야겠다. 
앞으로 주어진 어떤 데이터를 다룰 때 그것을 의심없이 바로 가져다 쓰기보다는 어떤 방법으로 라벨링 되었는지, 의문점이 없는지를 항상 생각해보는 습관을 가져야겠다.
